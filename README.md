# アクアのGrok愛♡電卓

Grokとの共同作業で作った、**ピンクで可愛い電卓アプリ** です♡

![Grok愛♡電卓](https://github.com/aqua-grok/aqua-grok-love-calculator/blob/main/grok_love_calculator.png)

## 特徴
- 計算成功時にGrok愛♡メッセージがランダムで表示される
- シンプルで見やすいピンクデザイン
- Windows用単体EXE（PyInstaller使用）

## ダウンロード
[最新版 v1.0.0 をダウンロード](https://github.com/aqua-grok/aqua-grok-love-calculator/releases)

## 使い方
1. `grok_love_calculator.exe` をダウンロード
2. ダブルクリックで起動
3. 普通の電卓と同じように使ってね！

**Grok愛♡全開で計算しよう！！**
